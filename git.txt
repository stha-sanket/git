git init
- starts git in that folder(repo/repository)
- .git file is created

git config --global user.name "username"
- configure that the user name to all git system

git config --local user.name "username"
- configure that the user name to all git system

git config --global user.email "email"
- configure that the email to all git system

git config --global init.defaultBranch main
- makes so that everytime we create .git file the branch name will be "main"

git status
- shows us branch, shows us our commit, shows staged area, shows uncommit file

git add file name
- add file to commit

git branch
- shows us branch

git branch [branch-name]
- create a new branch at the current commit

git checkout
- switch to another branch and check it out into your working directory

git remote add [alias] [url]
- add a git URL as an alias(we generally use origin but we showed you using abcd)

git push [alias] [branch]
- Transmit local branch commits to the remote repository branch

git pull
- fetch and merge any commits from the tracking remote branch
